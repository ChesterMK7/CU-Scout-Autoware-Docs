# Docker

[Docker](https://www.docker.com/) is a virtualization platform that utilizes *containers* to run software in a self-isolated environment. Containers are *similar* to Virtual Machines, however they often utilize cut-down OS images that lack a desktop environment and some usually-installed packages.

A more in-depth overview of Docker can be found [here](https://docs.docker.com/get-started/docker-overview/).
