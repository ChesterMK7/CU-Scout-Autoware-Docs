## Docs

Consult the Autoware docs for the docker install instructions and a general overview of how it all works. The tutorials section has useful information on setting up vehicle and sensor interfaces, as well as more technical information pretaining to CUDA and ROS2 development tools.

Note that we are utilizing Autoware within a Docker container, installed using the Autoware-provided installation tutorial, albiet with some slight modifications. The exact setup process can be found on the Setup Scripts page.
